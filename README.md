## Why is this on my github 
This is just going to be a quick experiment and demo on creating native-image with the latest micronaut and jdk/kotlin with the stuff you typically would see on a starter microservice

Is `./gradlew dockerBuildNative` going to mess with me? I'm testing to see if I could get a mildly complicated project to be compiled into native-image. At work, it sort of explodes when there is more than 5 controllers.
